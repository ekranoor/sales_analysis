Diwali Sales Analysis
 Project Overview
This project focuses on analyzing Diwali sales data to understand customer purchasing behavior, product performance, and regional sales trends using Exploratory Data Analysis (EDA).
The goal is to extract meaningful business insights that can help improve marketing strategies and sales planning during festive seasons.

 Key Insights
- Customers in the **26–35 age group** contributed the highest sales
- **Female customers** generated higher purchase value compared to males
- **Food, Clothing & Apparel, and Electronics** were the top-selling categories
- **Uttar Pradesh, Maharashtra, and Karnataka** recorded the highest sales
- Professionals from **IT and Healthcare** sectors showed higher spending
